# Valezap

## Sobre

Um sistema de tickets muito simples baseado em mensagens do WhatsApp, que permite multiusuários na mesma conta do WhatsApp.

## Requisitos

| Mínimo | Recomendado |
| --- | --- |
| Node 14.x | Node 16.x |
| Ubuntu 18.04 | Ubuntu 20.04 |
## Referência

- [Whaticket](https://github.com/canove/whaticket)
